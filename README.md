# 켜켜이 (Kyeokyeoi)
> 켜켜이 쌓인 추억을 한 권의 책으로

## 서비스 소개
"켜켜이"는 결혼을 앞둔 자녀가 부모님께, 또는 소중한 사람에게 인생 이야기를 담은 책을 선물할 수 있는 서비스입니다.
사진이 있으면 사진으로, 없으면 기억을 글로 남겨 — 켜켜이 쌓인 추억을 한 권의 실물 책으로 만들어드립니다.

### 타깃 고객
- 결혼을 앞두고 부모님께 특별한 선물을 드리고 싶은 자녀
- 소중한 사람의 인생 이야기를 책으로 남기고 싶은 누구나

### 주요 기능
- 시기별 타임라인 편집기 (사진 업로드 또는 텍스트 입력)
- 사진이 없는 시기는 텍스트로 기억을 기록
- Book Print API를 통한 실물 책 자동 생성
- 책 미리보기 및 배송 주문

### B2B 모델
청첩장 회사(바른손카드 등)가 파트너로 API 연동하여, 청첩장 고객에게 부가 서비스로 제공하는 B2B 모델을 상정합니다.

## 실행 방법

### 사전 준비
- Node.js 18 이상
- [Book Print API](https://api.sweetbook.com) Sandbox API Key

### 설치 및 실행
```bash
# 저장소 클론
git clone https://github.com/hoilove304-droid/kyeokyeoi.git
cd kyeokyeoi

# 패키지 설치
npm install

# 환경변수 설정
cp .env.example .env
# .env 파일에 API Key 입력

# 서버 실행
node server.js
```

브라우저에서 `http://localhost:3000` 접속

### 환경변수
```
SWEETBOOK_API_KEY=your_sandbox_api_key_here
SWEETBOOK_API_URL=https://api-sandbox.sweetbook.com/v1 
PORT=3000
```

## 사용한 API 목록

| API | 용도 |
|-----|------|
| `POST /books` | 새 포토북 생성 |
| `POST /books/{bookUid}/cover` | 표지 이미지 추가 |
| `POST /books/{bookUid}/contents` | 시기별 콘텐츠 페이지 추가 |
| `POST /books/{bookUid}/finalization` | 책 최종화 |
| `POST /orders/estimate` | 주문 가격 견적 조회 |
| `POST /orders` | 실제 주문 생성 |
| `GET /templates` | 템플릿 목록 조회 |

## 설계 의도

### 왜 이 서비스를 선택했는가
결혼을 앞둔 자녀가 부모님께 드릴 수 있는 가장 특별한 선물은 "함께한 시간"이라고 생각했습니다. 그런데 많은 가정에서 오래된 사진이 없거나 잃어버린 경우가 많습니다. 켜켜이는 사진이 없어도 기억만으로 책을 만들 수 있다는 점에서 차별화했습니다.

### 비즈니스 가능성
Book Print API의 B2B 모델과 정확히 맞닿아 있습니다. 청첩장 회사가 파트너로 연동하면, 청첩장을 주문하는 고객에게 "부모님께 드리는 가족 이야기 책" 서비스를 부가 상품으로 제공할 수 있습니다. 청첩장 시장의 고객은 이미 "특별한 날을 기념하는 것"에 관심이 높아 전환율이 높을 것으로 기대됩니다.

### 더 시간이 있었다면 추가했을 기능
- AI 이미지 생성: 사진 없는 시기에 텍스트를 기반으로 AI가 삽화 자동 생성
- 공유 기능: 가족 여러 명이 함께 편집하는 협업 기능
- 모바일 최적화: 스마트폰에서도 편리하게 사용할 수 있는 반응형 UI
